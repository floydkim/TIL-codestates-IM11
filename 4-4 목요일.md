# spread syntax
```javascript
var obj = {
  a: "a",
  b: "b"
};
var str = "asdfqwer";
var arr = [11,22,33,44];

var newObj = {
  ...obj,
  ...str,
  ...arr
}
console.log(newObj);
```
다음과 같은 결과가 나온다.
```
"0":  11
"1":  22
"2":  33
"3":  44
"4":  "q"
"5":  "w"
"6":  "e"
"7":  "r"
"a":  "a"
"b":  "b"
```
`...`로 펼치는 대상은 iterable이어야 한다.

```javascript
var newObj = {
  ...obj,
  ...str,
  ...arr
}
```

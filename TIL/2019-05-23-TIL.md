### Programmers
#### 정수 내림차순으로 배치하기
임의의 여러자리 정수가 주어졌을 때, 각 자리의 숫자를 내림차순으로 재 배치한 숫자를 리턴하는 함수를 작성한다.
```js
function solution(n) {
    var arr = String(n).split("");
    var sorted = arr.sort((a, b) => b - a);
    
    return Number(sorted.join(""));
}
```
